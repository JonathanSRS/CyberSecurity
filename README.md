# CyberSecurity

### Medusa
medusa -h {IP} -U {.txt} -P {.txt} -M {mod} -t {num} -T {num}\
-h Host\
-U Usuário ou lista de usuários\
-P Senha ou lista de senhas\
-M Modúlo\
-t Núméro de logins\
-T Número de hosts\

enum4linux -a {IP} | tree {output.txt}
